# ransom-demo-ui

Educational demo UI that switches between **Normal** and **Ransomware Incident** modes.
Safe-by-design: no remote execution, no collection agents, only mock data.

## Run locally

```powershell
npm install
npm run dev
```

Open: http://localhost:5173/#/dashboard

## Deploy to GitHub Pages (HTTPS)

Push to `main`, then enable:
Settings → Pages → Source = GitHub Actions
